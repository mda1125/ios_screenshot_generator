# iOS Screenshot Generator

A simple, browser-based tool for creating App Store-ready screenshots with device frames, headlines, and 3D perspectives.

## Features

- **Multiple Device Sizes** - iPhone 6.7", 6.9", 6.5", 6.1", 5.5" and iPad 13", 12.9", 11"
- **Text Overlays** - Add bold headlines and subheadings with adjustable font sizes
- **Device Angles** - Flat, tilted left/right, hero perspective, or custom 3D rotation
- **Background Options** - Solid colors, linear/radial/mesh gradients, and patterns
- **8 Gradient Presets** - Ocean, Sunset, Purple Haze, Midnight, Forest, Coral, Noir, Sky
- **Device Frame** - Optional frame with customizable color and shadow
- **Instant Preview** - Real-time updates as you adjust settings
- **One-Click Download** - Export as PNG at exact App Store dimensions

## Usage

1. Open `ios-screenshot-generator.html` in any modern browser
2. Upload your app screenshot
3. Customize text, colors, device angle, and background
4. Click "Download PNG"

No installation required. Works entirely offline.
